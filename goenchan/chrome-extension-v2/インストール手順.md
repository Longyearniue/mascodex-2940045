## 概要

Contact Form Auto-Filler Pro は、お問い合わせフォームを自動で入力するChrome拡張機能です。

**特徴:**
- ✅ **完全自動:** ページを開くだけで2秒後に自動入力
- ✅ **高精度:** 6層の検出システムで80-90%の入力成功率
- ✅ **日本語対応:** 日本のお問い合わせフォームに最適化
- ✅ **プライバシー保護:** データはローカルに保存、外部送信なし

---

## 主な機能

### 🎯 6層自動検出システム

1. **SITE_MAPPINGS** - 手動登録済みサイト（100%精度）
2. **Pattern Recognition** - 5種類のフォームパターン認識
3. **Auto-Generated** - Bulk Crawlerで自動生成（299サイト）
4. **Auto-Detection** - キーワードマッチング
5. **Semantic Analysis** - ラベル文字の意味解析（NEW）
6. **Generic Fallback** - 残りのフィールドを総当たり（NEW）

### 🚀 便利機能

- **自動入力:** ページ読み込み後2秒で自動的にフォーム入力
- **手動入力:** ボタン1クリックで即座に入力
- **Bulk Crawler:** URLリストから一括でフォームパターンを検出
- **Community Mappings:** 他のユーザーが検出したパターンを共有
- **Form Inspector:** フォーム構造を分析して手動マッピング作成

## インストール手順

### 方法1: ZIP解凍してインストール（推奨）

#### ステップ1: ZIPファイルを解凍

1. ダウンロードした `contact-form-auto-filler-pro-v2.20.0.zip` を**デスクトップ**などに解凍
2. 解凍後のフォルダ名: `contact-form-auto-filler-pro-v2.20.0`

**重要:** 解凍したフォルダは削除しないでください。拡張機能が動作しなくなります。

#### ステップ2: Chromeで開発者モードを有効化

1. Google Chromeを開く
2. アドレスバーに `chrome://extensions` と入力してEnter
3. 右上の **「デベロッパーモード」** トグルをONにする

#### ステップ3: 拡張機能を読み込む

1. 左上の **「パッケージ化されていない拡張機能を読み込む」** をクリック
2. 解凍したフォルダ `contact-form-auto-filler-pro-v2.20.0` を選択
3. 「フォルダの選択」をクリック

#### ステップ4: インストール完了を確認

拡張機能一覧に以下が表示されればOK：

```
Contact Form Auto-Filler Pro
バージョン: 2.20.0
```

## 初期設定

### プロファイル設定（必須）

拡張機能を使う前に、自分の情報を登録します。

1. **ポップアップを開く**
   - Chrome右上の拡張機能アイコン（パズルピース）をクリック
   - 「Contact Form Auto-Filler Pro」を選択

2. **Profile Settings を展開**
   - 「▶ Profile Settings」をクリック

3. **情報を入力**

   | 項目 | 例 |
   |------|-----|
   | Company / 会社名 | 株式会社サンプル |
   | Name / 氏名 | 山田太郎 |
   | Name Kana / フリガナ | ヤマダタロウ |
   | Email | yamada@example.com |
   | Phone / 電話番号 | 03-1234-5678 |
   | Zipcode / 郵便番号 | 100-0001 |
   | Address / 住所 | 東京都千代田区千代田1-1 |
   | Department / 部署 | 営業部 |
   | Subject / 件名 | お問い合わせ |
   | Message Template | お問い合わせ内容をご記入ください |

4. **自動入力を有効化**
   - ✅ **「🚀 ページ読み込み時に自動入力（完全自動）」** にチェック

5. **保存**
   - 「💾 Save Profile」をクリック

## 使い方

### 基本的な使い方（完全自動）

1. お問い合わせフォームのページを開く
2. **2秒待つ**
3. 自動的にフォームが入力される

**例:**
- http://www.yesp.co.jp/contact.html
- https://www.new-makomanai.com/inquiry/
- その他、どのお問い合わせフォームでも動作

### 手動入力（ボタンクリック）

自動入力が無効の場合、または再入力したい場合：

1. 拡張機能ポップアップを開く
2. 「✨ Auto Fill」ボタンをクリック
3. フォームが即座に入力される

### 動作確認（Console）

正しく動作しているか確認するには：

1. フォームページでF12キーを押す（DevToolsを開く）
2. 「Console」タブを選択
3. ページ読み込み後、以下のログが表示されればOK：

```
🚀 Auto-fill enabled for ALL sites. Starting auto-fill in 2 seconds...
⏰ Auto-fill timer triggered, calling autoFillForm...
✅ Auto-filled 6 field(s) automatically
📊 [SUMMARY] Layers used:
  - SITE_MAPPINGS: 0 fields
  - Pattern/Learned: 0 fields
  - Auto-detection: 4 fields
  - Semantic analysis: 0 fields
  - Generic fallback: 2 fields
  - TOTAL FILLED: 6/6 fields
```

## 高度な機能

### Bulk Site Crawler（一括フォーム検出）

複数のURLから一括でフォームパターンを検出します。

1. ポップアップで「▶ Bulk Site Crawler」を展開
2. URLを1行に1つずつ入力（例）:
   ```
   https://example.com/contact
   https://another-site.com/inquiry
   ```
3. 「🚀 Start Crawl」をクリック
4. 検出されたパターンは自動でchrome.storageに保存される

### Form Inspector（フォーム解析）

現在開いているフォームを詳しく分析します。

1. フォームページで拡張機能ポップアップを開く
2. 「🔍 このフォームを解析」をクリック
3. フィールド一覧が表示される
4. 必要に応じて手動マッピングを作成可能

### Community Mappings（共有パターン）

自動的に他のユーザーが検出したパターンをダウンロード・共有します。

- **自動ダウンロード:** ページ読み込み時に142個の共有パターンを取得
- **自動アップロード:** Bulk Crawlerで検出したパターンを自動共有
- **手動アップロード:** 「☁️ Share to Community」ボタンで手動共有

## トラブルシューティング

### フォームが入力されない

**原因1: プロファイル未設定**

→ 「初期設定」セクションを参照してプロファイルを登録

**原因2: 自動入力が無効**

→ ポップアップで「🚀 ページ読み込み時に自動入力」にチェック

**原因3: フォームが特殊**

→ DevToolsのConsoleでエラーメッセージを確認

### 間違った値が入力される

**原因: プロファイル設定の誤り**

例: 「Company」フィールドに名前が入っている

→ ポップアップでプロファイルを確認・修正

### 拡張機能が表示されない

**原因: デベロッパーモードが無効**

→ `chrome://extensions` で「デベロッパーモード」をON

### 拡張機能が動作しない

**原因: フォルダを削除した**

→ 解凍したフォルダを元に戻すか、再インストール

## よくある質問（FAQ）

**Q: データはどこに保存されますか？**

A: すべてのデータはChromeのローカルストレージ（chrome.storage.sync）に保存されます。外部サーバーには送信されません（Community Mappingsを除く）。

**Q: Community Mappingsは安全ですか？**

A: はい。共有されるのはフォームの「構造」（フィールド名、セレクター）のみで、あなたの個人情報（名前、メールなど）は送信されません。

**Q: 自動入力を無効化できますか？**

A: はい。ポップアップで「🚀 ページ読み込み時に自動入力」のチェックを外してください。

**Q: 特定のサイトで自動入力を無効化できますか？**

A: 現在はすべてのサイトで統一設定です。個別設定は今後のバージョンで検討します。

**Q: メッセージ欄に定型文を使いたい**

A: プロファイル設定の「Message Template」に定型文を入力してください。

**Q: バージョンアップ方法は？**

A: 新しいZIPファイルをダウンロードして、同じ手順で上書きインストールしてください。
