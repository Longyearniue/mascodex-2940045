【パッケージ内容】
chrome-extension-v2-package.zip

【機能】
✅ 問い合わせフォームの自動入力
✅ AI営業レター自動生成
✅ 完全自動モード対応
✅ 複数サイト対応

【インストール方法】

1. ZIPファイルを解凍
   chrome-extension-v2-package.zip → chrome-extension-v2フォルダ

2. Chromeを開く
   アドレスバーに「chrome://extensions/」と入力

3. デベロッパーモードをON
   右上のトグルをクリック

4. 拡張機能を読み込む
   「パッケージ化されていない拡張機能を読み込む」をクリック
   → chrome-extension-v2フォルダを選択

5. インストール完了！
